# Music playlist

## Описание

Проект создан для поиска песен на сайте [genius](https://genius.com/) по критериям:
* Имя автора
* ID артиста
* ID песни

Программа выводит:
* Название песни
* Имя музыканта
* Ссылку на текст
* Ссылку на изображение
* Дату выхода песни

Программа скачивает и закидывает в папку songs:
* Текст песни
* Обложку песни

## Установка

Используйте pip (или pip3, есть конфликт с Python2) для установки зависимостей:

```
pip install -r requirements.txt
```

## Поиск по имени автора

```python
python search.py --authorname Gazan
```
или
```python
python search.py -n Gazan
```
Данный поиск позволяет найти и скачать песни по имени автора. После аргумента указываем имя автора на английском.

В концоле выведится:
```
Название: С ДНЁМ РОЖДЕНИЯ (HAPPY BIRTHDAY)
Музыкант: Gazan
Ссылка на текст: https://genius.com/Gazan-happy-birthday-lyrics
Ссылка на изображение: https://images.genius.com/569eca350664ada113cd4c00db075274.1000x1000x1.png
Дата выхода: August 23, 2021
```
В папку songs скачаются все песни автора найденые на сайте [genius](https://genius.com/).

## Переменные окружения

Часть настроек проекта берётся из переменных окружения. Перменные окружения - это переменные, значения которые присваиваются программе Python извне. Чтобы их определить, создайте файл .env в корне и запишите туда данные в таком формате: ПЕРЕМЕННАЯ=значение.

Пример содержания файла .env:
```
API_KEY = "0f67e867867367067567b67f5267f672675b11b960221"
```

Получить ключ можно на сайте [genius](https://genius.com/).
